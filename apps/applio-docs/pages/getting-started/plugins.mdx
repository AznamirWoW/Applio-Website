# Plugins
### What are plugins in Applio?
Plugins or extensions, as the name implies, allow you to add functions to applio without having to update the program.

## How do I install plugins to Applio?
<Steps>
### Step 1: Download the plugins
Download the plugins you want from [here](https://github.com/IAHispano/Applio-Plugins/releases/latest).

### Step 2: Install the plugins
#### First Method
Go to the Plugins section and drag the plugin **.zip** file to Plugin Installer, when you upload the file and refresh the interface you will see the section of the extension you have downloaded.
![](/Plugin.png)

#### Second Method
Extract the plugin **.zip** file and move the extracted folder to `Applio/Tabs/Plugins/installed`.

</Steps>

## Update Plugin
Follow the previous same steps that you did to install the plugin.
