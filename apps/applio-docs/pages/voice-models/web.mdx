# Applio Web

If you are not closely related to discord or some other reason you can choose to use **[the web version of the Applio](https://applio.org/)**, search for any RVC voice model in our database.

![](/Applio-Website.png)

- Choose the model you wish to use and click on copy icon to get the link or on download icon to download it, then go to the [Download](/getting-started/download-models) tab of Applio.
